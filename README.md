# React Template RR

Modern React Router application template with Vite, TypeScript, and
comprehensive testing setup.

## Requirements

- Node 22+
- pnpm 9

## Installation

```bash
npx degit teo-garcia/react-template-rr my-app
cd my-app
pnpm install
pnpm dev
```

## Features

- **React Router** - Client-side routing
- **Vite** - Fast development and optimized builds
- **React** - Component-based UI library
- **TypeScript** - Static type checking
- **Vitest** - Unit testing with Testing Library
- **Playwright** - End-to-end testing
- **MSW** - API mocking
- **ESLint + Prettier** - Code quality and formatting
- **Husky + Lint Staged** - Pre-commit hooks

## Scripts

```bash
pnpm dev          # Start development server
pnpm build        # Production build
pnpm start        # Run production build
pnpm test:unit    # Run unit tests
pnpm test:e2e     # Run e2e tests
pnpm lint:es      # ESLint check
pnpm lint:ts      # TypeScript type check
pnpm format       # Format code with Prettier
```
